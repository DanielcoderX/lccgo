# lccgo

lccgo is a line and character counter.

## Installation

Just clone the repository and build it :)

```bash
git clone https://github.com/DanielcoderX/lccgo
cd lccgo/
go build -o build/lccgo
```

## Usage
Currently, support one subdirectory :(
```bash
lccgo ./main.go
lccgo src/
```
